> 결정 사항: 고객사 평가지표 요약수치만 공개 · 레드팀 QA 편집 · 비용분석 고객사 비공개 · 보존기간 7-1에서 설정

---

## RBAC 역할 정의

| 역할 | 설명 |
|------|------|
| **Admin** | 전체 접근. 권한 설정 가능 |
| **Operator** | 운영사 내부 엔지니어. 시스템 설정·모니터링 |
| **QA** | 품질 평가·교정·시뮬레이션 |
| **Customer** | 고객사 담당자. Admin이 허용한 메뉴만 접근 |

## 1. 대시보드

**목적:** 접속 즉시 "지금 이상 없나?" 30초 파악. 읽기 전용. 상세는 각 메뉴 링크로 이동.

### 1-1. 통합 관제

| 항목 | 내용 | 클릭 시 이동 |
|------|------|-------------|
| KPI 카드 5개 | E2E Latency P95 · Error Rate · Answer Rate · Index Freshness · Cost/Query | 각 카드 → 5-3 이상 징후 감지 |
| 파이프라인 레이턴시 바 | Retrieval / Reranking / LLM / 후처리 단계별 ms | — |
| 시스템 상태 신호등 | 정상 / 경고 / 위험 자동 판정. 하위 지표 중 가장 심각한 수준 기준 | — |
| 활성 알림 배너 | 미해소 알림 목록. 전체 페이지 상단 고정 노출 | 알림 항목 → 해당 지표 상세 |

- **Customer:** 접근 불가

### 1-2. 서비스 통계

| 항목 | 내용 | 클릭 시 이동 |
|------|------|-------------|
| DAU / 총 질의 수 | 일별 추이, 시간대별 사용량 | — |
| 세션 성공률 | 월별 추이, 목표값 대비 현황 | — |
| 질의 카테고리 분포 | 도넛 차트 | — |
| Knowledge Gap Rate | 답변 불가 비율(%), 미처리 건수 | 미처리 건수 → 4-2 미해결 질의 |

- **Customer:** DAU · 세션 성공률 · 카테고리 분포만 노출. Knowledge Gap Rate 수치 비공개

### 1-3. 품질/보안 요약

| 항목 | 내용 | 클릭 시 이동 |
|------|------|-------------|
| Faithfulness / Hallucination Rate | 현재값 + 7일 스파크라인 | → 4-1 평가 지표 |
| PII 감지 건수 | 이번 달 누적, 마지막 감지 시각 | → 5-2 보안 감사 로그 |
| 사용자 피드백 | 👍/👎 비율, 주간 추이 | → 4-3 답변 교정 |

- **Customer:** Faithfulness · Hallucination 수치만 노출. 스파크라인 · PII · 피드백 비공개

## 2. 지식베이스 관리

**목적:** 문서 업로드 → 전처리 → 인덱싱 파이프라인 전체 관리.

### 2-1. 데이터 업로드

| 항목 | 내용 | 클릭 시 이동 |
|------|------|-------------|
| 파일 업로드 | HWP / PDF / XLSX / DOCX. 드래그앤드롭 + 일괄 업로드 | — |
| 웹 크롤링 설정 | URL 등록, 주기(일/주), 제외 경로 지정 | — |

| 수동 재인덱싱 | 즉시 트리거 버튼. 스텝 인디케이터(업로드 → 전처리 → 인덱싱) 표시 | — |

### 2-2. 데이터 전처리

| 항목 | 내용 | 클릭 시 이동 |
|------|------|-------------|

| PII 비식별화 규칙 | 감지 엔티티 유형 선택, 마스킹 방식 설정. 저장 시 기존 인덱스 소급 적용 여부 확인 모달 | — |
| Ingestion Error 로그 | 파싱 실패 목록 · 오류 유형 · 재처리 버튼 | — |
| 문서 중복 감지 | 코사인 유사도 0.95↑ 청크 목록. 삭제/병합 처리 | — |

|------|------|-------------|

## 4. 품질 관리 (QA)

**목적:** 답변 품질 측정·개선. PRD v4 지표들이 집중 배치되는 섹션.

### 4-1. 평가 지표

| 항목 | 내용 | 클릭 시 이동 |
|------|------|-------------|
| 지표 목록 | Faithfulness · Hallucination Rate · Recall@K · Answer Relevance · Citation Correctness · Session Success Rate | — |
| 현재값 + 버전 비교 | 현재 배포 버전 vs 이전 버전 기본 뷰 | — |
| 이상 시 드릴다운 | 목표값 미달 지표 클릭 → 해당 샘플 케이스 모달 | — |

- **Customer:** 각 지표의 현재 수치만 노출. 버전 비교 차트 · 샘플 드릴다운 비공개

### 4-2. 미해결 질의

| 항목 | 내용 | 클릭 시 이동 |
|------|------|-------------|
| 질의 목록 | Fallback 질의. 카테고리 자동 분류 · 발생 건수 · 첫 발생일 | — |

| 지식베이스 추가 | 해당 질의 주제 문서 추가 바로가기 | → 2-1 데이터 업로드 |

### 4-3. 답변 교정 (Feedback)

| 교정 이력 | 교정자 · 일시 · 변경 전/후 기록 |

### 4-4. 시뮬레이션 룸

| 항목 | 내용 |
|------|------|
| 테스트 질의 입력 | RAG 응답 + 검색 청크 목록 + 유사도 점수 확인 |

| 결과 저장 | 저장 및 공유 링크 생성 |

### 4-5. 레드팀 케이스셋

| 항목 | 내용 | 클릭 시 이동 |

## 5. 서비스 모니터링

**목적:** 운영 이슈 추적 + 보안 감사. 대시보드의 "요약 + 링크" 대비 "상세 + 필터".

### 5-1. 대화 이력 조회

| 항목 | 내용 |
|------|------|
| 세션 로그 | 고객사별 격리. Customer는 자사 세션만 조회 가능 |
| 필터 | 기간 · 고객사 · 세션 성공여부 · 카테고리 |
| 세션 상세 | 청크 출처 · 유사도 점수 · 응답 시간 포함. Customer에게는 유사도 점수 비공개 |

- **QA:** 전체 세션 조회 가능하나 세션 상세 내 청크 출처 비공개

### 5-2. 보안 감사 로그

| 항목 | 내용 | 클릭 시 이동 |
|------|------|-------------|
| PII 감지 이벤트 | 감지 유형 · 마스킹 처리 여부 · 세션 ID. 원문 비공개 | → 2-2 PII 비식별화 규칙 |
| 관리자 접근 이력 | 시스템 접근 기록 | — |
| 금칙어 차단 로그 | 차단 건수 · 패턴 | — |

- **QA · Customer:** 접근 불가

### 5-3. 이상 징후 감지

| 항목 | 내용 |
|------|------|
| Query Drift | 7일 rolling 기준 이탈 추이 차트 |
| Recall Baseline Deviation | 7일 rolling 대비 이탈량 |
| Embedding Drift | 주간 분포 이동 |
| 비정상 반복 질의 | DDoS 의심 패턴 감지 |
| 임계값 설정 | 지표별 Warning/Critical 수치 직접 수정. Slack · PagerDuty 알림 연동 (→ 7-2와 연동) |

- **Customer:** 접근 불가

## 페이지 간 이동 흐름

```
[1-1 대시보드] KPI 이상 감지
  └─→ [5-3 이상 징후 감지] 원인 파악
        ├─→ 인덱스 문제 → [2-3 벡터 DB 관리]
        └─→ 품질 문제  → [3-1 프롬프트] 롤백

[1-2 서비스 통계] Knowledge Gap 미처리 건수 클릭
  └─→ [4-2 미해결 질의] 목록 확인
        └─→ [2-1 데이터 업로드] 문서 추가

[3-1/3-2] 프롬프트·파라미터 변경
  └─→ [4-4 시뮬레이션 룸] 변경 효과 검증
        └─→ [4-5 레드팀] 배포 전 안전성 확인
```
